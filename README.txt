학교숲 시민과학자 v16

수정 사항
1. GPS 정확도 초과 시 '지도에서 위치 지정' 화면의 배경 지도가 보이지 않던 문제를 수정했습니다.
2. 문제가 있던 두 번째 Leaflet 지도는 삭제했습니다.
3. 이미 정상 표시되는 '조사 이동 경로' OpenStreetMap 지도를 위치 지정에 재사용합니다.
4. 지도 중앙에 빨간 십자표시를 띄우고 지도를 움직여 위치를 지정합니다.
5. 화면 하단에 위치 지정 도구막대를 표시합니다.
6. 우측 상단에 닫기(✕) 버튼을 추가했습니다.
7. '측정 GPS' 버튼으로 원래 측정 위치로 돌아갈 수 있습니다.
8. '이 위치로 지정'을 누르면 종 기록 화면으로 돌아가 선택 좌표를 사용합니다.

확인 순서
- GPS 정확도 10m 초과
- 지도에서 위치 지정
- 조사 이동 경로 지도가 정상 표시되는지 확인
- 지도 이동
- 닫기 버튼으로 취소 후 종 기록 화면 복귀 확인
- 다시 위치 지정 후 '이 위치로 지정' 확인
