학교숲 시민과학자 v18

수정 사항
1. 학생 화면의 기록 탭에서 CSV 내보내기 버튼을 제거했습니다.
2. 학생 화면의 기록 탭에서 JSON 백업 버튼을 제거했습니다.
3. 관련 학생용 내보내기 함수와 이벤트 연결도 함께 제거했습니다.
4. 데이터 내보내기 기능은 추후 교사 화면에만 제공합니다.

학생 기록 탭에는 다음만 남습니다.
- 최종 제출 기록 목록
- 제출 상태
- 조사 결과 요약
- 제출 내용 보기
